# リリースプロセス

## リリース日

定期リリースは**火曜と木曜**の週 2 回です。金曜のリリースは原則禁止です（週末の障害対応を避けるため）。

## 段階的リリース

新機能は feature flag の背後で実装し、まず社内ユーザーのみに公開します。問題がなければカナリアリリースとして全ユーザーの **10%** に展開し、24 時間監視した後に 100% へ展開します。

## ロールバック基準

カナリア期間中にエラー率が通常時の **2 倍**を超えた場合、即座にロールバックします。判断に迷う場合はロールバックを優先します（Roll back first, debug later）。

## ホットフィックス

SEV1/SEV2 障害の修正は定期リリースを待たず、オンコール担当の承認でいつでもリリースできます。
