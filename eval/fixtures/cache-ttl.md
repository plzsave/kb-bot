# 回答キャッシュ運用ガイド

kb-bot は同一の質問に対する回答を SQLite に保存し、次回以降は LLM を呼ばずに即座に返します（回答キャッシュ）。これにより API コストと応答時間を大きく削減できます。

## 有効期限（TTL）について

回答キャッシュは**無期限**で保持されます。いちど生成された回答は自動的には失効せず、明示的にキャッシュを削除しない限り永続的に再利用されます。TTL による自動失効の仕組みは導入していません。

そのため、ドキュメントやコードを更新した後にキャッシュが古い回答を返し続ける場合は、運用者が手動でキャッシュ行を削除して再生成させてください。

## キャッシュの保存先

回答キャッシュはナレッジ索引と同じ `kb.sqlite` に同居しています。完全一致（正規化後）での照会にヒットした場合のみキャッシュを返します。
