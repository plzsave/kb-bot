# 認証とアクセス管理

## 認証方式

社内サービスへのログインは Google アカウントによる OIDC で行います。社外アカウントではログインできません。
API アクセスには発行された Bearer トークンを `Authorization` ヘッダに付与します。

## トークンの発行

設定画面の「API トークン」から発行できます。トークンは発行時に一度しか表示されないため、必ず控えてください。
有効期限は既定で 90 日です。失効した場合は再発行が必要です。

## 権限レベル

- 閲覧者: ナレッジの参照のみ
- 編集者: ナレッジの追加・更新
- 管理者: メンバー管理とトークン失効

権限の変更は管理者に依頼してください。
